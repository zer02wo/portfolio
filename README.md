# portfolio
A place for me, my thoughts and my personal projects.

Built in Astro, TypeScript and Tailwind.

## Local Development
### Initialise Docker:
```
docker compose up --build
```

### Start Docker:
```
docker compose up
```

### Stop Docker
```
docker compose down
```

### Update package-lock.json
```
docker compose run web npm install
```
